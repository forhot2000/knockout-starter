# knockout-starter

knockout 项目的启动包，支持编译和发布功能。

## install

```shell
npm install --save-dev @forhot2000/knockout-starter
```

```yarn
yarn add -D @forhot2000/knockout-starter
```

modify package.json

```json
{
  "scripts": {
    "build": "knockout-starter build",
    "start": "knockout-starter start"
  }
}
```

## build

```shell
npm run build
```

or

```shell
yarn build
```

## start

```shell
npm start
```

or

```shell
yarn start
```
